# d2cpp | 动手学C++

一个基于[xlings](https://github.com/d2learn/xlings)构建且强调动手实践的C++学习项目

## 使用项目

> 需要[安装xlings环境](https://github.com/d2learn/xlings)

### 下载

> 下载d2cpp到当前目录

```bash
xlings drepo d2cpp
```

注: `xlings drepo`可以查看项目信息

### 打开项目书籍

```bash
xlings book
```

### 运行项目练习

> 运行d2cpp项目中的cpplings代码练习

```bash
cd d2cpp
xlings dslings
```
